University Database System

This project is a Python and SQLite-based University Database System that demonstrates how to manage student records, course enrollments, and relational data using SQL. It includes a CLI interface for creating students, enrolling in courses, withdrawing, and viewing enrollments. The project highlights practical database skills, SQL JOINs, CRUD operations, and error handling.

Features:

Create new student records

Enroll students in courses

Withdraw students from courses

List student’s enrolled courses using SQL JOINs

Search courses by name

CLI-based user interface

Demo Database:

new_sql.db contains sample data (5 students, 5 courses, enrollments)

Small database included for demonstration purposes

Setup:

1. Clone the repository

git clone https://github.com/issa920920/university_database_system.git

cd university_database_system

2.Download the new_sql.db file from this repo and make sure it’s in the same folder as university_db.py

3.Run the Python script to start the CLI:

python university_db.py


4.Follow prompts to create new students or interact with sample data 

Usage:

C: Create a new student

M: List your enrolled classes

A: List all available courses

E: Enroll in a course

W: Withdraw from a course

S: Search courses by name

X: Exit
